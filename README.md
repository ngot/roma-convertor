###Roma Cconvertor
#####*Roma number convert tool*

###Installation

Install with [npm](http://npmjs.org):

```bash
  npm install roma-convertor --save
```

###Usage
```js
  var rome = require('roma-convertor');
  var m = roma.parse(1);
  var n = roma.depParse("V");
```
